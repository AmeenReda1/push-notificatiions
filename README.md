# Push Notification Service

This is a NestJS project designed to send push notifications using Firebase Cloud Messaging (FCM) when a new product is added.

## Overview

This service is built using NestJS to handle push notifications for a product-based application. Whenever a new product is added to the system, a push notification is sent to all relevant users via Firebase Cloud Messaging (FCM).
